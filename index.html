Tutorial Sabun 10 Langkah

  1. Siapkan dulu bahannya, lalu copy file dan folder, seperti gambar dibawah

    *file bisa didownload disini : kreasi sabun

    *file yang saya gunakan di tutorial ini sudah saya sedikit modifikasi dari file yang diberikan pa Ramdhani

  2. Pastekan file dan folder ke folder project tugas ksk kalian

  3. Lalu edit file service.php di code editor favorit kalian (notepad++, sublime, VSCode, dsbg)

  4. pada Fungsi connectDB() ubah host, username, password, nama database, dan pada Fungsi getTb() ubah nama table sesuai dengan pengaturan database kalian

    *host, username dan password biasanya sama dengan gambar diatas jika masih pengembangan di localhost (ubah jika sudah di deploy / diupload / di hosting ke server kalian)

  5. pada Fungsi getTb() ubah sesuai dengan struktur table kalian

    *gambar diatas hanyalah contoh, struktur table bisa kalian liat di localhost/phpmyadmin

  6. pada Fungsi insertTb() ubah parameter fungsi, field dan nama_tabel sesuai dengan struktur table kalian.

  7. pada Fungsi updateTb() lakukan hal yang hampir sama seperti pada langkah 5, perhatikan gambar

    *untuk lebih cepat, parameter fungsi bisa kalian copy & paste dari code parameter insertTb()

    *bedanya dengan langkah 5: (nama_field1, nama_field2) VALUES ('param1', 'param2'), langkah 6: nama_field1 = 'param1'

    *contoh lain : nama_karyawan = '". $nama_karyawan ."', gaji = '". $gaji ."'

  8. pada Fungsi deleteTb(), kalian cuma perlu mengubah 'id' (setelah WHERE, jika field primary key kalian berbeda dengan gambar) sesuai dengan nama field id / primary key di struktur table kalian.

    *biasanya field id adalah field yang paling awal dan primary key, contoh : id_barang, kode_barang, nip, nik, dsbg
